# Photo Sensor Input - Discrete Signal Processing
A discrete time filtering of photo-sensed input to analyze relative motion.


### Overview
Detect direction of motion by using two photo-sensors.


### Problem
Visualization of data from simple detectors. 


### Equipment Used
- Arduino Uno
- 2x photoresistors
- Breadboard/Wires
- MATLAB


### Setup
1. Wire the two photoresistors to the Arduino
2. Setup MATLAB to take input from Arduino
3. Upload the driver code

Contribution: Circuit setup and debugging
